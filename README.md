# Qt C++ Custom Calculator

# Calculator is composed of three screens Calculator, Recent, Extra. 

# 1. Calculator follows order of operations going from Parthensis, Multiplication and Division, to Subtraction and Additon.
#    As well it has ^2 button, percentage, sqr(), log base 10 and inverse
#    **Small easter egg bonus ON button flashes background colors"
# 2. Recent holds the last 10 searches in the calculator and when selected can pull them back into the calculator to be modified
#    and used agian. As well clear all clears all values in the calculator
# 3. Extra holds more hard coded variables as well as a button to allow the user to change the background color of the calculator.

# Enjoy :)
